# PHP Listmonk API

Implementation of the Listmonk API in PHP. It requires PHP >= 7.4.

## Install

Packagist is not available yet, so use composer vcs.

```json
{
  "repositories": [
    {
      "type": "git",
      "url": "https://github.com/junisan/php-listmonk.git"
    }
  ],
  "require": {
    "junisan/php-listmonk": "^0.1"
  }
}
```
